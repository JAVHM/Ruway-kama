Para hacer una prueba se debe ejecutar el siguiente comando en la consola
--->direccion/backend
npm install
luego
-->direccion/backend
node index.js

Para las vistas de (VerProyecto,dashboard,CrearProyecto, Mensajeria)
localhost:puerto/VerProyecto
localhost:puerto/dashboard
localhost:puerto/mensajeria
localhost:puerto/crearProyecto
El resto de vista funciona con los botones, "inicar sesion", "registro", "login"
